About EyeCloud

This website is designed for visual computing project course (2IMV15) at Technology University of Eindhoven (TU/e). The aim of this project is to help public transport map designers and producers to visualize and analyse eye movement data tracked from study participants. https://github.com/veneres/EyeCloud
